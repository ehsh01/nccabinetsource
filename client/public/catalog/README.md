# Catalog Images

Add kitchen/bathroom photos here for each cabinet style and color. When a user clicks a color swatch, the corresponding image will display.

## File Naming

Use lowercase, hyphen-separated filenames. Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`

### Brooklyn Series
| Color | Filename |
|-------|----------|
| Bright White | `brooklyn/bright-white-1.jpg` … `brooklyn/bright-white-7.jpg` |
| Midnight | `brooklyn/midnight-1.jpg`, `brooklyn/midnight-2.jpg`, `brooklyn/midnight-3.jpg` |
| Modern Gray | `brooklyn/modern-gray-1.jpg` … `brooklyn/modern-gray-6.jpg` |
| Slate | `brooklyn/slate-1.jpg`, `brooklyn/slate-2.jpg`, `brooklyn/slate-3.jpg` |
| Fawn | `brooklyn/Fawn-1.jpg` … `brooklyn/Fawn-5.jpg` |

### Shaker Series

### Oslo Series
| Color | Filenames (use multiple for collage) |
|-------|-------------------------------------|
| White | `oslo/white-1.jpg`, `oslo/white-2.jpg`, `oslo/white-3.jpg` |
| Oak | `oslo/oak-1.jpg`, `oslo/oak-2.jpg`, `oslo/oak-3.jpg` |

## Adding Images in Cursor

1. Drag and drop images into the appropriate folder (e.g. `client/public/catalog/brooklyn/`)
2. Use the exact filenames from the table above
3. Images should show a kitchen or bathroom with that cabinet style and color
